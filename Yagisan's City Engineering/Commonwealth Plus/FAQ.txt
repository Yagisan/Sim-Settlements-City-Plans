[font=Verdana][color=#00ff00][size=5][b]Frequently Asked Questions[/b][/size][/color]

[color=#6d9eeb][b]Why don't my plots look like yours?[/b][/color]
Please make sure you have all the addon packs installed, and that designers's choice is set to "on" in the SimSettlements options before you build your city.

[color=#6d9eeb][b]Why am I missing crafting stations?[/b][/color]
Please make sure that you have [url=https://bethesda.net/en/mods/fallout4/mod-detail/3411824]CREAtive Clutter[/url] installed, and then refresh your city plan.

[color=#6d9eeb][b]Why didn't any city plan bugfixes appear?[/b][/color]
You need to refresh bugfixed city plans before the fixes will appear.

[color=#6d9eeb][b]How do I refresh a city plan?[/b][/color]
First select the blueprint on the desk, and scroll down to [b]Refresh City[/b]. Select it. It may take some time.
Second, activate the terminal, select Tools -> Configuration Tools -> [b]Rebuild Local Data[/b]. Select it. It may take some time.

[color=#6d9eeb][b]The city has advanced a level, and now objects are spawning inside each other. How do I fix this?[/b][/color]
Please remain in the settlement for a few minutes after a level upgrade.
It takes some time for the deletion code to run after an upgrade. Larger settlements take longer to finish.
